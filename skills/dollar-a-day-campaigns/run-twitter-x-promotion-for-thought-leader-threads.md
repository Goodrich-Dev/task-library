# Run Twitter/X promotion for thought leader threads

**Use this when** a thread or key tweet has clearly outperformed the account's baseline and the author's authority should compound beyond their followers.

## Inputs
- The account's recent threads/tweets with metrics (impressions, replies, reposts, bookmarks, profile clicks)
- X Ads account under the owner's control, payment set, website tag installed if traffic is the goal
- The peer-account list: whose audiences should see this thinking

## Steps
1. Identify the organic winners with the standard signal read: threads and tweets with outsized bookmarks, replies, reposts, and profile clicks **relative to the account's own baseline** — the /dad discipline applied to X.
2. Verify the plumbing: X Ads account owned by the thought leader (not an agency), billing live, conversion/website tag in place if clicks matter to the Goal.
3. Write the GCT: Goal (reach decision-makers, grow follows, or drive traffic), Content (the proven thread — promote its first tweet so readers enter at the top), Targeting (follower-lookalikes of named peer accounts, interests, keywords, geo).
4. Promote the **existing post**, never a re-upload — replies and reposts ride along as social proof, the same rule as switch boosts.
5. Structure one audience per ad group at minimum daily spend — the Dollar a Day philosophy of many small simultaneous tests — and run a full 7-day window untouched.
6. Judge by cost per engagement plus profile-visit and follow lift, not raw impressions. Kill ad groups below target; scale winners ≤2× per adjustment; rotate the next proven thread in.
7. Flag the gap while you work: this play has no definitive article yet — log learnings and screenshots in the test log as raw material for the future hub.

## Definition of done (QA checklist)
- [ ] Only baseline-beating threads promoted, entered from the first tweet of the thread
- [ ] Original post promoted with social proof intact; one audience per ad group at minimum spend
- [ ] 7-day window held; cost per engagement and follow lift scored against a written target
- [ ] Learnings logged toward the missing definitive article (gap flagged)
- [ ] Linked back to the definitive article and relevant siblings
- [ ] Complies with Blog Posting Guidelines (if it publishes content)

A persistent agent (Claude Fable 5 or comparable OpenAI/Google models) runs the X loop whole — baseline read, GCT, promote the original first tweet, 7-day window, verdicts — and only stops when the Definition of done fully passes; an ad group judged early or left running with no kill/scale verdict is a failed run by the Dollar a Day rule.
